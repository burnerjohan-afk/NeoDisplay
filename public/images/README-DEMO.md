# Image pour la démo Smartfilm

## Image nécessaire

Pour la démonstration interactive du Smartfilm, vous devez ajouter une image de salle de réunion vitrée.

### Fichier à ajouter

- **Nom** : `salle-reunion-vitree.jpg` (ou `.webp`, `.png`)
- **Emplacement** : `public/images/salle-reunion-vitree.jpg`
- **Taille recommandée** : 1200x900px (ratio 4:3)
- **Poids** : < 500KB (compressez si nécessaire)

### Caractéristiques de l'image

L'image doit montrer :
- Une salle de réunion ou un bureau moderne
- Des cloisons vitrées (partitions en verre)
- Des fenêtres extérieures
- Un intérieur professionnel

### Utilisation

Une fois l'image ajoutée, elle sera automatiquement utilisée dans le composant de démonstration interactive.

Pour utiliser une image différente, modifiez le chemin dans `components/smartfilm/Demo.tsx` ligne 40.

### Sources d'images libres de droits

- Unsplash : https://unsplash.com (recherchez "glass office", "meeting room glass", "modern office")
- Pexels : https://www.pexels.com (recherchez "glass partition office")
- Pixabay : https://pixabay.com

### Note

Actuellement, le composant utilise une image Unsplash en placeholder. Remplacez-la par votre image locale pour de meilleures performances.
